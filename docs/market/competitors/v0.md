# v0

## Company Overview

Built by Vercel as an AI UI generation tool. Not a standalone company (~$3.25B Vercel valuation). 6M+ developers and 80K+ active teams. Tightly integrated with the Next.js/Vercel ecosystem.

## Product/Service Offering

Converts text prompts into production-ready React/Tailwind components. Frontend-first -- no backend generation. Deep integration with Next.js and Vercel deployment.

## Pricing Strategy

Feb 2026: switched from fixed-credit to token-based pricing (caused backlash). Free: $5/mo in credits (~7-15 generations). Paid: $10-20/mo.

## Marketing & Messaging

UI generation for the React/Next.js ecosystem. Positioned as the fastest path from idea to deployed frontend component.

## Market Perception

### Strengths

- Clean, idiomatic React output widely praised
- Deep Vercel/Next.js integration
- Large developer reach (6M+)

### Weaknesses

- Next.js lock-in limits appeal outside Vercel ecosystem
- No backend generation
- Feb 2026 pricing change caused user backlash
- Non-developers misuse it expecting a Figma-like visual editor
