# Replit Agent

## Company Overview

Cloud IDE that pivoted to AI in 2024. $253M ARR late 2025 (16x YoY from $16M). $250M raised at $3B valuation (Sep 2025). 40M registered users, 150K+ paying customers, 750K+ businesses.

## Product/Service Offering

Autonomous agent inside a cloud IDE: writes, runs, debugs, and deploys full-stack applications. Supports 50+ languages with built-in PostgreSQL, hosting, and secrets management. Agent 3: autonomous for up to 200 minutes, capable of building other agents.

## Pricing Strategy

Effort-based pricing. Simple tasks under $0.25, larger tasks metered. Core plan: ~$25/mo + $25 in credits.

## Marketing & Messaging

Zero-setup, most capable platform for full-stack development. From idea to deployed app without leaving the browser.

## Market Perception

### Strengths

- Zero-setup experience praised as best-in-class
- Most capable platform for full-stack app generation
- Massive user base (40M registered)
- Strong revenue growth (16x YoY)

### Weaknesses

- Pricing opaque and anxiety-inducing for users
- Agent 3 needs manual intervention on edge cases
- Large free-tier user base (learners) makes monetization challenging
