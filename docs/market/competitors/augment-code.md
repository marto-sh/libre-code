# Augment Code

## Company Overview

Founded Apr 2024 by Igor Ostrovsky (ex-Microsoft) and Guy Gur-Ari (ex-Google AI). $252M total funding, backed by former Google CEO Eric Schmidt. ISO/IEC 42001 certified -- the first AI coding assistant to achieve this. Positions itself as "The Software Agent Company."

## Product/Service Offering

**Context Engine:** Live semantic index of entire codebase (400K+ files) with millisecond sync. "Memories" feature for cross-session persistence.

**Inline completions:** "Next Edit" predictive editing.

**Augment Chat:** Conversational coding assistant.

**Augment Agent:** Multi-step autonomous task execution.

**Remote Agents (Summer 2025):** Cloud-based parallel execution.

**Code Review Agent (Dec 2025):** Highest public benchmark accuracy at 65% precision.

Default model: Claude Sonnet 4.5. Certifications: SOC 2 Type II, ISO 42001, CMEK.

## Pricing Strategy

| Tier | Cost | Credits |
|------|------|---------|
| Indie | $20/mo | 40K |
| Standard | $60/user/mo | 130K |
| Max | $200/user/mo | 450K |
| Enterprise | Custom | Custom |
| Auto top-up | $15 | 24K credits |

Credit consumption: simple task ~300 credits, complex task ~4,300 credits. Oct 2025: switch to credit-based pricing caused backlash, with reports of 2-3x cost increases for power users.

## Marketing & Messaging

"Build software with AI agents that understand your entire codebase." "Context, not completion, is what makes code enterprise-ready." Security-first and agent continuity positioning. Targets mid-to-large engineering teams.

## Market Perception

### Strengths

- Most context-aware product for large codebases (400K+ files)
- ISO 42001 certification is unique in the space
- 65% code review precision leads public benchmarks
- Responsive support team
- Remote Agents well-received
- High Gartner Peer Insights ratings

### Weaknesses

- Oct 2025 pricing backlash ("completely misleading" credit model)
- $60/dev/mo expensive vs Copilot at $19/user/mo
- Performance issues (VS Code slowdowns, responses hanging 2-3+ min)
- Struggles with niche or less common libraries
- Still requires significant human oversight
