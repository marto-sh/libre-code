# Bolt

## Company Overview

Built by StackBlitz, launched Oct 2024. Leverages WebContainers technology (Node.js running in-browser). Reached $20M ARR in 60 days and $40M by Mar 2025. ~35 employees, 5M+ users. Described as "second fastest-growing product after ChatGPT."

## Product/Service Offering

Full-stack app generation from natural language, running entirely in-browser. Bolt V2/Cloud adds databases, authentication, file storage, and Netlify hosting. No local setup required.

## Pricing Strategy

Token-based. Free: 1M tokens/mo. Paid tiers: $20-200/mo.

## Marketing & Messaging

Zero-setup, instant full-stack apps. Browser-native development removes all environment friction.

## Market Perception

### Strengths

- Fast revenue growth ($40M ARR by Mar 2025)
- Large user base (5M+)
- Code quality solid for small projects
- Zero-setup browser experience

### Weaknesses

- Quality degrades past 15-20 components
- 31% success rate on enterprise-grade features
- In-browser architecture limits scalability for complex apps
