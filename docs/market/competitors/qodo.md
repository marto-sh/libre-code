# Qodo

## Company Overview

Founded 2022 in Israel by Itamar Friedman (ex-Alibaba) and Dedy Kredo (ex-VMware). Rebranded from CodiumAI in 2024. $50M total funding: $11M seed (2023) and $40M Series A (Sep 2024, Susa Ventures, Square Peg). ~100 employees. 1M+ developers have used its tools. Named a Gartner Visionary in 2025.

## Product/Service Offering

"Agentic code integrity platform." Core products:

- **PR-Agent:** Automated code review with 15+ specialized sub-agents running in parallel per quality dimension
- **Qodo Gen:** Test generation (original product)
- **Qodo Cover:** Test coverage analysis
- **Context Engine:** Multi-repo understanding (Enterprise)

Available as IDE plugin (VS Code, JetBrains), CLI for CI/CD, and SaaS. Deployment options: SaaS, VPC, and air-gapped. Sub-2-minute PR scans.

## Pricing Strategy

| Tier | Cost | Limits |
|------|------|--------|
| Developer (Free) | $0 | 30 PRs/mo, 75 credits |
| Teams | $30/user/mo | Unlimited PRs, 2,500 credits |
| Enterprise | Custom | SSO, on-prem, multi-repo, SLA |

## Marketing & Messaging

"Deploy with confidence." Quality-first positioning: other tools generate code, Qodo ensures correctness. "Code integrity." Focuses on review debt elimination and enterprise safety. Framed as complementary to code generation tools rather than a replacement.

## Market Perception

### Strengths

- Gartner Visionary recognition
- Sub-2-minute scan times are fast
- High-signal review comments (precision over noise)
- Multi-repo understanding for enterprise
- TechCrunch AI Disruptors 60 list
- Free tier drives evaluation pipeline

### Weaknesses

- Agentic mode buggy (Python indentation errors reported)
- Slower on large contexts
- Lacks architectural-level insight compared to CodeRabbit
- Learning curve for enterprise configuration
- Limited brand recognition outside developer communities
- Niche risk as general-purpose tools improve their review capabilities
