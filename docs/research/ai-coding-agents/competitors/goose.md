# Goose

## Company Overview

Open-source AI agent by Block (formerly Square). Announced Jan 2025. Apache 2.0 license. 27K+ GitHub stars, 350+ contributors, 100+ releases. Dec 2025: contributed to Linux Foundation's Agentic AI Foundation (AAIF) alongside MCP and AGENTS.md. Runs a Goose Grant Program offering up to $100K for open agentic AI projects.

## Product/Service Offering

Terminal + desktop agent with full autonomy (reads/writes files, runs code, executes tests, calls APIs). Model agnostic (Claude, GPT-4, Gemini, local via Ollama) with multi-model support within a single session. Native MCP support (3,000+ servers). Extension types: built-in, remote (HTTP/SSE), and command-line. "Distro" customization allows forking and rebranding. Local-first by design -- code stays on device.

## Pricing Strategy

Entirely free. No subscription, no freemium, no markup. Strictly BYOK. Free LLM options available via Google Gemini, Groq, and Ollama. Block funds development through its corporate OSPO.

## Marketing & Messaging

"Open source, extensible AI agent beyond code suggestions." "On-machine AI agent." "Works with any LLM." "Build your own goose distro." Positioned as infrastructure layer rather than product. Linux Foundation governance emphasized for long-term credibility.

## Market Perception

### Strengths

- Genuinely free with zero markup
- Model neutrality valued by multi-provider teams
- Strong community growth (27K+ stars, 350+ contributors)
- AAIF governance gives institutional credibility
- Architecturally sound local-first design
- Grant program incentivizes ecosystem development

### Weaknesses

- Setup complexity for non-technical users
- Terminal-first friction limits adoption
- No subsidized inference (users must bring their own keys)
- Desktop/web UI less polished than CLI experience
- Safety risks with full autonomy mode
- Output quality entirely LLM-dependent
