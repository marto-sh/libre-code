# Cline

## Company Overview

Created by Saoud Rizwan, launched 2024 as a hackathon project. Based in Sacramento, CA. Raised $32M combined Seed + Series A (Jul 2025) led by Emergence Capital and Pace Capital. Apache 2.0 license. 50K+ GitHub stars, 5M VS Code installs, ~20K Discord members. Originally named "Claude Dev" (package.json still references claude-dev).

## Product/Service Offering

VS Code (and JetBrains) autonomous agent with Plan/Act mode (approval before execution). MCP integration (3,000+ servers; can self-generate MCP servers). Subagents via CLI instances. Headless CLI 2.0 (Feb 2025): stdin/stdout piping, `-y` flag for full autonomy, CI/CD compatible. Checkpoints and diff review. Browser automation (headless). Model agnostic (BYOK or managed provider).

Enterprise features: SSO, SCIM, OIDC, audit logs, VPC deployment, OpenTelemetry observability.

## Pricing Strategy

| Tier | Cost | Details |
|------|------|---------|
| Individual | Free | BYOK, pay provider directly |
| Teams | Free Q1 2026, then free up to 10 seats | $20/user/mo beyond 10 seats |
| Enterprise | Custom | SSO, SCIM, VPC, self-hosted |

## Marketing & Messaging

"Open source and uncompromised." "Human in the loop every step." "Your infrastructure, your inference." Privacy-first positioning. Contrasted against subscription tools with markup. Enterprise customers include Samsung and SAP.

## Market Perception

### Strengths

- Strong cross-file reasoning quality
- Community trust through full source availability
- Strongest forks (Roo Code, Kilo Code) validate the architecture
- Excellent MCP ecosystem integration (3,000+ servers)
- 50K+ stars and 5M installs demonstrate adoption

### Weaknesses

- Raw API costs steep ($50/day or $200/mo on heavy Claude usage)
- Innovation pace slower than its own forks
- Checkpoint limitations (early 2025)
- Browser automation struggles with heavy SPAs
- Setup friction from BYOK configuration
- Step-by-step execution lacks automatic re-verification
